# Shivani's Portfolio Website

A modern, responsive portfolio website built with HTML, CSS, and JavaScript to display your resume in an attractive application format.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Interactive Navigation**: Tab-based navigation for easy browsing between sections
- **Animated Elements**: Smooth animations and transitions throughout the site
- **Print-Friendly**: Download/print resume functionality built-in
- **Modern UI**: Beautiful gradient design with professional styling

## Sections

1. **About** - Personal information and introduction
2. **Experience** - Work history with timeline visualization
3. **Education** - Educational background and certifications
4. **Skills** - Technical skills with visual progress bars
5. **Projects** - Portfolio of completed projects

## How to Use

1. Open `index.html` in any modern web browser
2. Navigate through different sections using the tab buttons
3. Customize the content with your own information
4. Click "Download Resume" to print or save as PDF

## Customization

### Update Personal Information

Edit the following in `index.html`:

- **Profile Picture**: Replace the placeholder image URL in the `<img>` tag
- **Name & Title**: Update the `<h1>` and `<p>` tags in the profile section
- **Contact Info**: Modify email, phone, and location
- **Social Links**: Add your actual LinkedIn, GitHub, and portfolio URLs

### Modify Content

- **About Section**: Update the biography and personal details
- **Experience**: Add/remove job entries in the timeline
- **Education**: Update educational background
- **Skills**: Adjust skill levels by changing the percentage in `style="width: X%"`
- **Projects**: Add your own projects with descriptions and tags

### Styling

Modify `styles.css` to change:
- Color scheme (search for `#667eea` and `#764ba2` to change the gradient)
- Fonts and sizes
- Layout and spacing
- Animations and transitions

### Functionality

Edit `script.js` to:
- Add custom interactions
- Integrate with external APIs
- Add form functionality
- Enhance animations

## Browser Compatibility

## Tips

- Use high-quality images for best results
